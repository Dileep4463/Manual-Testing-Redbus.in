 Manual Testing Project – Redbus.in

Project Overview
Performed manual testing on the **Redbus.in** web application to verify functional and UI features across modules like login, bus search, seat selection, and payment gateway.

Deliverables
- **Test-Plan.xlsx** – Defines objectives, scope, and strategy.
- **Test-Cases.xlsx** – Contains functional and regression test cases.
- **Bug-Report.xlsx** – Lists identified defects with severity and priority.
- **RTM.xlsx** – Maps requirements to test cases for 100% coverage.
- **Screenshots/** – Evidence of test case execution.

Tools & Techniques
- **Tools:** Excel, JIRA  
- **Testing Types:** Functional, Regression, Usability, Smoke Testing  

Outcome
- Validated all critical workflows in the Redbus web application.  
- Practiced complete **STLC** including test design, execution, and bug reporting.  
- Improved understanding of real-time web application testing scenarios.
